# 酒店管理系统 java945

## 项目概述

酒店管理系统是一款采用前后端分离架构设计的系统，基于 Java 和 Vue 技术栈开发。此系统致力于实现酒店业务的数字化管理，提高酒店运营效率。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows/Mac
- **开发环境**:
  - IDEA
  - JDK1.8
  - Maven
  - Mysql
- **技术栈**:
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue
